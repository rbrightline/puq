src/lib/names/names.ts

names
names(resourceName: string, options?: NamesOption)

Generates various case formats for a given resource name.

Parameters :
Name Type Optional Description
resourceName string No

the resource name
options NamesOption Yes

extra options such as prefix, suffix NamesOption
Returns : Names

An object containing the resource name in different case formats:

  • camelCase: camelCase format.
  • constCase: CONSTANT_CASE format.
  • kebabCase: kebab-case format.
  • pascalCase: PascalCase format.
  • snakeCase: snake_case format.
  • titleCase: Title Case format.
  • controllerName: NameController.
  • serviceName: NameService.
  • moduleName: NameModule.

src/lib/common/normalize.ts

normalize
normalize(value: string, preInputValidator?)

Normalizes a given string by adding spaces before uppercase letters and capitalizing the first letter. Optionally, a custom pre-input validator function can be provided to validate the input string.

Parameters :
Name Type Optional Description
value string No
  • The string to be normalized.
preInputValidator Yes
  • An optional function to validate the input string before normalization. If not provided, a default validation is applied which checks if the string contains only letters, underscores, dollar signs, and spaces.
Returns : string

The normalized string with spaces before uppercase letters and the first letter capitalized.
