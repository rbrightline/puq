dirs(root: string, options?: CommonFileOptions)

Asynchronously retrieves an array of directory paths starting from a root directory. Can operate recursively and return either absolute or relative paths based on options.

Parameters :
Name Type Optional Description
root string No
  • The root directory path to start the search from
options CommonFileOptions Yes

Returns : Promise<string[]>

A promise that resolves to an array of directory paths
