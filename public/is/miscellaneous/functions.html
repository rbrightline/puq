src/lib/object/deep-clone.ts

deepClone
deepClone(obj: T)

Creates a deep clone of an object, preserving its structure and values without shared references. This function recursively copies all properties, including nested objects, arrays, and special types like Date, while handling circular references to prevent infinite recursion. The clone is fully independent, ensuring no side effects on the original object.

Parameters :
Name Type Optional
obj T No
Returns : T
  • {T} A deep-cloned copy of the input object.
